Testing support for code snippets:

Ruby
```ruby
def print_hi(name)
  puts "Hi, #{name}"
end

print_hi('Tom') # Hi, Tom
```

Javascript
```javascript
function print_hi(name) {
  console.log("Hi, #{name}")
}

print_hi('Tom') // "Tom"
```

HTML
```html
<html>
hi
</html>
```
